# 🏆 COPAFAN.APP

> O hub definitivo para acompanhar a Copa do Mundo FIFA 2026 com transmissões integradas e estatísticas em tempo real.

Disponível em: **[https://gabrielcaetanodev.github.io/copafan.app/](https://gabrielcaetanodev.github.io/copafan.app/)**

---

## ⚽ Funcionalidades Principais

* **📺 CazéTV Integrada**: Assista às transmissões ao vivo direto do site com painéis inteligentes (contagem regressiva para futuros jogos e melhores momentos para jogos encerrados).
* **📊 Estatísticas Estilo Google**: Minuto a minuto detalhado com comentários dos lances em português, barras de posse de bola, finalizações, faltas e cartões.
* **📋 Campinho Tático**: Visualize a escalação de cada seleção distribuída taticamente em um campo de futebol, além da lista completa de jogadores reservas.
* **🌳 Chaveamento Interativo**: Acompanhe o mata-mata a partir da Fase de 32 até a Final no MetLife Stadium com zoom, arraste e iluminação de trajetória dos times ao passar o mouse.
* **📱 Split Screen de Seleções**: Busque qualquer uma das 48 seleções e confira histórico de jogos (Vitórias/Empates/Derrotas coloridos), elencos oficiais e estatísticas agregadas.
* **⚙️ Motor de Simulação Híbrido**: Funciona de forma autônoma simulando jogos ao vivo ou conectando diretamente à API do `football-data.org` salvando a chave localmente.

## 🛠️ Tecnologias Utilizadas

* **React 19** & **TypeScript**
* **Vite** (Build e bundling rápido)
* **Tailwind CSS v4** (Visual escuro premium com efeitos de glassmorphism)
* **Lucide React** (Ícones modernos)
* **GitHub Actions** (Deploy automático contínuo)

---

## 🚀 Como Executar Localmente

1. Instale as dependências:
   ```bash
   npm install
   ```
2. Execute o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
3. Para compilar o projeto para produção:
   ```bash
   npm run build
   ```
